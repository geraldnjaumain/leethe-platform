# Leethe Task Handoff Protocol

## Instructions for Agent
Upon completion of any implementation step or task:
1. Update [`SCOPE.md`](file:///Users/tera/Documents/leethe/SCOPE.md) progress status table if milestones changed.
2. Append a new entry to [`ITERATION_LOG.md`](file:///Users/tera/Documents/leethe/ITERATION_LOG.md) detailing:
   - **Completed in this iteration**: Files modified/created, test results, visual verification output.
   - **Scope Alignment Check**: Explicit confirmation of zero scope creep.
   - **Next Task Specification**:
     - Objective.
     - Files to create or modify next.
     - Skills to activate.
     - Expected outputs and verification criteria.
